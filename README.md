## 运行

### 环境要求

- Node.js 23+
- npm 或 yarn

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

打开 [http://localhost:3000](http://localhost:3000) 查看应用。

### 构建生产版本

```bash
npm run build
```

重要提示：本项目仅供学习研究使用，严禁商业用途！
